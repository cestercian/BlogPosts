# BlogPosts

A simple blog post application that allows users to view, create, and manage blog posts.
This project uses the JSONPlaceholder API
for testing and provides features like fetching, displaying, and adding posts.

## Features

- Fetch and display blog posts from JSONPlaceholder API.
- Add new blog posts via a simple form.
- Dynamic updates to the blog list upon submission.

## Technologies Used

- **HTML** for the structure of the page.
- **CSS** for basic styling.
- **JavaScript** for fetching data and handling form submissions.
- **JSONPlaceholder** for mock data.


### 2. Install Dependencies

Since this is a front-end project, you don't need to install server-side dependencies,
but you can use **`Live Server`** in your editor for a quick preview
or configure any local server to serve the files.

### 3. Open the Project

Open the project folder in your favorite text editor, such as **WebStorm**.

### 4. Run the Project

Simply open the `index.html` file in your browser or run it through any local server setup.
